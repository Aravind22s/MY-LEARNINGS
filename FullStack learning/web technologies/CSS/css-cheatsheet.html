position cheat guide
staticDefault flow. top/left/z-index ignored.
relativeOffset from its normal position. Creates stacking context.
absoluteRemoved from flow. Positioned relative to nearest non-static ancestor.
fixedRemoved from flow. Positioned relative to viewport.
stickyStays in flow until scroll threshold, then fixed.
